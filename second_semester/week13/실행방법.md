# Week 12 실행 방법

## 1. 가상환경 생성 및 패키지 설치

```bash
cd /Users/kyeongbin/codyssey/ia-codyssey/second_semester/week12

# 가상환경 생성 (처음 한 번만)
python3 -m venv venv

# 가상환경 활성화
source venv/bin/activate

# 패키지 설치
pip install -r requirements.txt
```

## 2. 서버 실행

```bash
cd /Users/kyeongbin/codyssey/ia-codyssey/second_semester/week12

# 가상환경 활성화 (매번 실행 전에)
source venv/bin/activate

# 서버 실행
python3 -m uvicorn main:app --reload
```

또는:

```bash
uvicorn main:app --reload
```

## 3. API 테스트

서버가 실행되면 다음 주소로 접속:

-   **Swagger UI (API 문서)**: http://localhost:8000/docs
-   **질문 목록 API**: http://localhost:8000/api/question/
-   **루트 엔드포인트**: http://localhost:8000/

## 4. 문제 해결

### Import 오류가 발생하는 경우

week12 폴더에서 실행해야 합니다:

```bash
cd /Users/kyeongbin/codyssey/ia-codyssey/second_semester/week12
python3 -m uvicorn main:app --reload
```

### 패키지가 설치되지 않는 경우

```bash
pip3 install --upgrade pip
pip3 install fastapi uvicorn sqlalchemy pydantic
```

### 데이터베이스 오류가 발생하는 경우

데이터베이스가 없으면 자동으로 생성됩니다.
만약 문제가 있다면:

```bash
# 데이터베이스 초기화 (필요시)
rm board.db
python3 -c "from database import engine, Base; from models import Question; Base.metadata.create_all(bind=engine)"
```

## 5. API 엔드포인트 확인

Swagger UI (http://localhost:8000/docs)에서 다음을 확인:

-   ✅ GET `/api/question/` - 질문 목록 조회 (새로 추가된 라우터)
-   GET `/` - 루트 엔드포인트
-   POST `/questions` - 질문 생성
-   GET `/questions` - 모든 질문 조회
-   GET `/questions/{question_id}` - 특정 질문 조회
